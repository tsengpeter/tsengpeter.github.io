Hexo 是什麼

Hexo 是一個基於 node.js 框架基礎下產生的快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他標記語言）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態網頁檔案。

前置作業

前置安裝

在開始使用 Hexo 之前，電腦必須先安裝以下工具:

  • 安裝 node.js : 使用 npm 指令來安裝所需的套件

    Hexo 官網建議使用 Node.js 10.0 及以上版本，若不確定下載哪個版本，可以直接下載最新的 node 版本。

  • 安裝 git : 使用 git 來上傳到 github 空間，以此發布 github page

    git 指令請上網找，在此不教學 git 相關的詳細教學

hexo 操作時常用指令

新增文章:

  • 如果沒有設定 layout，則會使用 _config.yml 中的 default_layout 來設定
  • 如果標題有包含空格，需使用引號括住，例如”title”
  • 接著可直接到 /source/_posts/title.md 中編輯文章內容
  • 檔案名稱盡量以英文命名，避免出現亂碼

清除快取:

在每次儲存修正後的檔案之前，會建議先輸入此指令，清除快取檔案 （db.json）和已產生的靜態檔案（public）。

產生靜態檔案:

可簡寫成 hexo g，從 hexo 的 markdown 檔案產生成靜態網頁檔案。

啟動伺服器:

  • 在本地端啟動 Hexo 伺服器，預設路徑為：http://localhost:4000/
  • 可在自己電腦上預覽設定結果，按 Ctrl + C 即可關閉

    localhost：代表只能從本地瀏覽此網站，無法從外部瀏覽

可簡寫成 hexo s

Hexo 主題(theme)更換

Hexo 最新版本的主題(theme)預設是使用 landscape 這個主題，如果想要更換主題(theme)，可以到Hexo 官網的 theme挑選妳覺得好看的主題(theme)，通常你想要的主題(theme)附上的 github 連結會有使用教學，怎麼將主題匯入到你的 hexo 專案裡，大致上的步驟指令為下:

P.S 慎選主題，有些主題可能太舊沒維護，或是教學寫得完全看不懂

Step1: 開啟終端機

開啟終端機，並將終端機輸入指令的路徑改到 hexo 專案裡

到 hexo theme 裡面挑選一個你看上的主題(theme)，然後點到主題附上的 github 內，然後 copy 這個 github 專案的連結

部屬到 github 上

建立 github 專案

這邊將以用 github pages 來作為架設 hexo 的部落格網站地方。

接著就照著以下的步驟一一實作:

Step1: 註冊 github 帳號並且登入，

Step2: 登入 github 網站後，在畫面的左上方會有一顆綠色的 new 按鈕，點選 New 新增一個 Repository（專案）

Step3: 將專案名稱命名為 username.github.io，username 選到你自己，這樣就成功建立了一個網域：username.github.io
