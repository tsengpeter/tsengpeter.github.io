什麼是 Eslint

Eslint 是一個靜態程式碼分析工具，用於識別和修復 JavaScript 程式碼中的問題。它可以幫助開發者保持一致的程式碼風格，並避免常見的錯誤。

什麼是 Prettier

Prettier 是一個程式碼格式化工具，用於自動調整程式碼的格式，使其符合指定的風格規範。它可以與 Eslint 一起使用，以確保程式碼既正確又美觀。

什麼是 Husky

Husky 是一個 Git hooks 工具，用於在特定的 Git 事件（如 commit 或 push）發生時，自動執行指定的腳本。它可以用來強制執行程式碼檢查和格式化，確保程式碼庫的品質。

  • eslint-config-prettier: 當你使用 ESLint 和 Prettier 時，有些 ESLint 規則（例如格式相關的規則）會與 Prettier 格式化衝突（如縮排、引號類型等）。這個套件會自動禁用這些規則，讓 Prettier 接管格式化部分。

  • eslint-plugin-prettier: 讓 Prettier 成為一個 ESLint 插件，將 Prettier 的格式化錯誤當作 ESLint 錯誤報告出來。
